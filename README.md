#webrtc.io-demo
==============

You can have a look at the [demo](http://webrtc.dennis.is/)

##Instructions on how to setup the demo:

Run in your terminal

```bash 
git clone git@github.com:webRTC/webrtc.io-demo.git
```

```bash 
cd webrtc.io-demo
```

```bash 
npm install
```

```bash 
cd site
```

```bash 
node server.js
```
